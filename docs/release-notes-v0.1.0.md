# GhostCapture v0.1.0

First public Windows installer release of GhostCapture.

Highlights:

- compact WPF launcher for Android mirroring
- USB device detection and one-click start flow
- Wi-Fi debugging QR pairing inside the main window
- dedicated ADB server runtime to avoid clashes with other ADB clients
- low-latency `scrcpy` launch path with no console window
- installer packaging through Inno Setup

Download:

- `GhostCapture-Setup.exe`
